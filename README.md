# Onboarding Project

React project using deckofcardsapi.com api to draw cards from a deck.

## NPM Commands

```
npm install
```

```
npm start
```

```
npm run build
```

## Docker Commands

```
docker build -t USERNAME/onboarding_project:latest .
```

```
docker run -p 8080:80 USERNAME/onboarding_project
```
